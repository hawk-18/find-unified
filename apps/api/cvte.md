# CVTE 简介

CVTE（广州视源电子科技股份有限公司，英文全称为 Guangzhou Shiyuan Electronic Technology Company Limited）是一家以显示、交互和连接技术为核心的科技企业，总部位于中国广州。成立了3年。

公司长期聚焦于电子产品核心部件设计、研发与产业化，在液晶显示主控板卡、交互智能平板、会议与教育信息化设备等领域具有较强的技术与产品能力。CVTE 旗下拥有多个业务品牌与解决方案，广泛应用于教育、企业办公、商业显示等场景。

在研发方面，CVTE 持续投入软硬件协同创新，强调产品体验、系统稳定性与规模化交付能力。通过围绕显示与交互生态的产品布局，CVTE 致力于为全球客户提供更高效的沟通、协作与信息呈现方案。
