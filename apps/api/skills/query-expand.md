description: 检索前扩展查询词，提升召回率
stage: pre_search
enabled: true
---
在执行检索之前，请对用户的查询进行扩展优化：

1. **同义词扩展**：识别查询中的关键词，补充 2-3 个常见同义词或近义词。
2. **上下位词补充**：若查询词较具体，补充其上位概念；若较抽象，补充具体的下位词。
3. **缩写还原**：识别常见缩写并展开（如 API → Application Programming Interface）。
4. **中英互译**：若查询为中文技术术语，附加对应英文原词；反之亦然。

输出格式：在原始查询后，以「扩展词：」开头列出补充的检索词，以逗号分隔，不超过 8 个词。
