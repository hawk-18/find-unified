收到检索结果后，在生成回答前请对证据片段进行重排序：

1. **相关性优先**：将与查询意图最直接相关的片段排在最前。
2. **时效性**：若多个片段内容相似，优先使用来源路径中包含更新日期的片段。
3. **去冗余**：若多个片段内容高度重复（>70% 重叠），只保留得分最高的一条。
4. **权威来源加权**：来自官方文档、README、CHANGELOG 的片段权重高于博客或注释。

重排后，使用排序后的证据列表作为回答依据。
